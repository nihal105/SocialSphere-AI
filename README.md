##  Project Overview

**SocialSphere AI** is an end-to-end Machine Learning project that leverages **K-Means Clustering** to identify hidden behavioral patterns within Facebook users. By analyzing engagement, friendships, activity levels, and interaction metrics, the model automatically segments users into meaningful groups for data-driven insights and decision-making.

This project demonstrates the complete Data Science workflow—from data preprocessing and exploratory analysis to model development and deployment using Streamlit.

---

##  Objective

The primary goal of this project is to:

- Identify distinct Facebook user segments
- Analyze user engagement patterns
- Discover influencers and highly active users
- Support targeted marketing strategies
- Enable personalized user experiences

---

##  Dataset Features

The model analyzes user behavior using the following attributes:

| Feature | Description |
|----------|-------------|
| Age | User age |
| Tenure | Days spent on Facebook |
| Friend Count | Total number of friends |
| Friendships Initiated | Friend requests sent |
| Likes | Total likes given |
| Likes Received | Total likes received |
| Mobile Likes | Likes from mobile devices |
| Mobile Likes Received | Likes received via mobile |
| WWW Likes | Likes from web platform |
| WWW Likes Received | Likes received via web |

##  User Segments Identified

### 🌱 New Users
Users with lower engagement and fewer social connections.

### ⭐ Influencers
Users receiving significant engagement and audience interaction.

### 👴 Senior Users
Long-term users with stable activity patterns and extensive platform experience.

### 👑 Celebrity Users
Highly popular users with exceptional reach and visibility.

Users who consistently interact, engage, and expand their social network.
